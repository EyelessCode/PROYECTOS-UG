Tecnologías Cliente

  • Las tecnologías cliente trabajn del lado del browser, se clasifican:
  • HTML - Se encarga de la estructura, contenido y visualización.
    • HTML
  • CSS - Se encarga del diseño visual para atractividad.
    • CSS
  • JavaScript - Interacción y lógica para que el usuario pueda interactuar.
    • JAVASCRIPT

Protocolo HTTP

¿QUÉ ES?

HTTP define un conjunto de métodos de petición para indicar la acción que se desea realizar para un recurso determinado.

  1. GET
  2. HEAD
  3. POST
  4. PUT
  5. DELETE
