Tecnologías del lado del cliente

HTML

HTML, creado por Tim Berners-Lee a finales de los años 80, permitió compartir documentos científicos en la web. HTML 1.0, lanzado en 1993, estableció las bases del lenguaje. HTML evolucionó rápidamente con HTML 2.0 en 1995 y HTML 3.2 en 1997, que introdujeron más capacidades. En 1997, HTML 4.0 permitió la creación de páginas más dinámicas. XHTML, lanzado en 2000, intentó llevar más rigidez a la web usando XML, pero no fue ampliamente adoptado. HTML5, iniciado en 2008 y lanzado oficialmente en 2014, trajo importantes mejoras como multimedia y elementos semánticos, reflejando la realidad de la web moderna. Hoy en día, el desarrollo de HTML continúa bajo el modelo de actualización continua del WHATWG.

CSS

CSS (Cascading Style Sheets) se desarrolló para separar la presentación del contenido en la web. Fue propuesto por Håkon Wium Lie en 1994, mientras trabajaba en el CERN, y su primera versión oficial, CSS1, fue lanzada en 1996 por el W3C. CSS permitió a los desarrolladores aplicar estilos a las páginas web de manera más sencilla y consistente. En 1998, se lanzó CSS2, introduciendo características como posicionamiento absoluto y relativo, y soporte para media queries. Posteriormente, CSS3 se dividió en varios módulos para facilitar su desarrollo y adopción gradual, trayendo mejoras significativas como transiciones, animaciones y más. Hoy en día, CSS sigue evolucionando para adaptarse a las necesidades de la web moderna, permitiendo diseños responsivos y dinámicos.

JavaScript

JavaScript fue creado por Brendan Eich en 1995 durante su tiempo en Netscape Communications. Originalmente llamado Mocha, el lenguaje fue rápidamente renombrado a LiveScript y, finalmente, a JavaScript para aprovechar el creciente interés en Java. JavaScript se convirtió en un componente esencial de la Web, permitiendo interactividad en las páginas. En 1997, ECMAScript se estandarizó como la versión oficial de JavaScript bajo ECMA-262. Las versiones sucesivas, como ES5 en 2009, introdujeron características importantes, y ES6 (o ECMAScript 2015) en 2015 trajo mejoras significativas como clases, módulos y funciones de flecha. JavaScript sigue evolucionando, con actualizaciones anuales que introducen nuevas funcionalidades y mejoras, manteniéndolo como un pilar fundamental de la programación web moderna.
