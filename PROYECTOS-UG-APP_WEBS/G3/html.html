HTML Características

Técnica empleada

La técnica empleada en este proyecto es la de desarrollo web, utilizando HTML como lenguaje de marcado para la estructura de la página web.

Función

La función de HTML es la de estructurar la página web, utilizando etiquetas para organizar el contenido y hacerlo más legible y fácil de entender.

Etiqueta

Las etiquetas en HTML son utilizadas para definir el contenido de la página web, como títulos, párrafos, enlaces, imágenes, etc.

Semántico

HTML es un lenguaje de marcado que se utiliza para estructurar y organizar el contenido de una página web. Es el lenguaje de marcado más comúnmente utilizado en la creación de sitios web.
